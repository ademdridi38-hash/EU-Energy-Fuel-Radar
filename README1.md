# 🇪🇺 EU Smart Intelligence Radar
A Python-based automated monitoring system for European markets.

### 🚀 Features:
- **Energy Tracker:** Monitors real-time electricity prices in Germany/EU.
- **Fuel Sniper:** Tracks E5/Diesel price fluctuations.
- **Flight Hunter:** Notifies about low-cost flight deals across Europe.
- **Telegram Integration:** Instant alerts with statistical analysis.

### 🛠 Tech Stack:
- Python (Asyncio, Requests)
- Telegram Bot API
- Open Data APIs (ENTSO-E, Tankerkoenig)
